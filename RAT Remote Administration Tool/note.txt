suffix 3333 ------------ to start a program in shell
suffix 1111 ------------ to change computer name
suffix 2222 ------------ nick name in chat
suffix 4444 ------------ play sound
suffix 5555 ------------ double click time
suffix 6666 ------------ set cursor position
suffix 7777 ------------ play a beep
suffix 8888 ------------ terminate 
suffix 9999 ------------ message

suffix x... ------------ to have full text length, in cursor
